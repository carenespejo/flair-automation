# FLAIR Selenium Automation Framework

## Overview
This repository contains the **Selenium-based QA automation framework** for **Project FLAIR**.  
It is built using **Python, Pytest, and the Page Object Model (POM)**, with additional **workflow abstraction** to represent real business processes such as sourcing, approval, and publishing.

The framework is designed to be:
- Scalable
- Maintainable
- CI-ready
- Easy to understand for both technical and non-technical stakeholders
